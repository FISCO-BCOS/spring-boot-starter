# 贡献代码

非常感谢能有心为FISCO-BCOS社区贡献代码！

## 代码格式化

执行任务 `googleJavaFormat`格式化java文件。
```
./gradlew goJF
```
执行任务 `verifyGoogleJavaFormat`验证java文件是否格式化完成
```          
./gradlew verGJF
```
